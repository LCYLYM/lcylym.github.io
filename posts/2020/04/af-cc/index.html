五分钟安装教程

  • 1.安装条件  本程序分为主控和被控,主控安装仅需要支持php且可以访问的空间/服务器/虚拟主机即可!,主控仅需上传3个文件就可以完成安装 主控最低硬件配置:随便,能运行php就行被控需要Centos7系统且有独立公网ip(ipv6未经测试不确定支持,不支持nat云/共享ip云)的云服务器/vps/cvm/ecs/服务器(暂不支持其他但您可以自行尝试或修改安装源码),我们提供了一键安装指令,您只需要输入指令其他什么都不需要即可完成安装 被控最低硬件配置:建议至少1核512Mb以上(但实际上并没用任何关系) 温馨提示:华为腾讯阿里百度京东云以及有安全组的云请提前开放10086端口! 本程序除了接口程序外,全部开源!全部免费!
    安装被控请注意一下,如果你的这台服务器是装了宝塔/amh/kangle等面板可以不影响原有服务的情况下直接使用,直接安装,无需全新服务器,但如果您之前就是用源码建站了,可能会冲突!

2.被控安装 

  • 使用SSH,以root用户进入您的centos7的ssh终端,随后输入一键安装指令,五分钟完成安装在终端输入:
yum -y install wget && wget -O wb.sh https://loli.by/file/wb.sh && chmod +x wb.sh && sh wb.sh
  • 随后输入1然后回车即可安装过程中可能系统会询问,全部选择y即可完成安装本安装程序将会在您的系统中安装caddy php7.0 webbench 不会占用太多资源(其实吧纯程序60Mb内存就可以了)请注意,当被控重启之后,程序并不会开机自启,请输入sh wb.sh 随后输入3并回车以启动程序服务!
  • 3.主控安装  主控就是php程序,您只需要点我下载源码,然后上传到网站/虚拟主机,访问即可!主控的所有程序都为您写好了,您只需要在bk.txt文件中输入您的被控的公网ip(或域名)即可!您必须要在bk.txt中添加至少一个被控IP才可以访问主控!!!您想要使用集群进行cc攻击时,您只需要在所有的发包机/被控机按照2.被控安装安装被控程序,随后将ip写入bk.txt即可,请注意一行一个,不要留空格!不要使用Windows的记事本编辑!!!不然会出错!!!
  • 4.其他说明  版权请不要删,毕竟免费开源了,很多地方可以自己修改也可以自己增加功能,甚至可以对接一些销售系统~不要用做违法内容以及^_^基于webbench基本秒杀无防的网站,但对于有防御的束手无策~请注意,被控安装的环境不要给别的程序使用,极度不安全!!!!!!
